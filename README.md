# SimplePasswordGen
## Simple Password Generator
Greg McIntyre  
24/10/2019  
  
Generates 10 Passwords, copies selected to clipboard.  
  
  
### Prerequisite
```
$ sudo apt install xsel
$ sudo apt install python3-pip
$ sudo pip3 install pyperclip
```
  
### Run
```
SimplePasswordGen/PasswordGen.py
```
